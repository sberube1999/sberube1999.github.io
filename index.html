Ubisoft Game Lab 2026

Lunar Lounge

A competitive intergalactic server simulation developed for the Ubisoft Game Lab competition. Our team completed this project within a 10-week development cycle while balanced against a full-time University schedule.

Main Programming Contributions

Physics & Locomotion

Developed acceleration-based movement with velocity-dependent turn rates to achieve a fluid "rollerblades" game feel. Developed custom grounded/slope-handling logic and physics-driven mechanics including dashing, stuns, and directional bumps.

Gameplay Systems

I made a modular event system, including dynamic spawning of food hazards triggered by clients actions and time-based power-up spawning. Developed the logic and mechanics for all in-game hazards and player buffs.

Technical Tools & more

Built a custom Debug Command Tool to accelerate internal testing and managed the Lobby logic.
